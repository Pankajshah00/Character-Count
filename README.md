# Character-Count
Character Count Web App

A simple character counter built using HTML, CSS, and JavaScript. This tool helps users quickly count characters in a given text input.

Features

Real-time character count

Word count display

Supports spaces and special characters

Responsive design

Technologies Used

HTML

CSS

JavaScript

Usage

Type or paste text into the input box.

The character and word count will update in real-time.

Clear the input to reset the count.
